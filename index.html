Automatic categorization Python script

The script reads a transaction file, applies categorization rules (based on keywords in descriptions) and generates an enriched file with business categories.In this a case I just used a simple python if-else function. For a big dataset like yours we will use Transformers (the main technology behind ChatGPT) to classify each transaction.
